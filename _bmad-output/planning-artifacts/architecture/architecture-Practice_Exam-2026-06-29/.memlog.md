- (decision) Paradigm: modular monolith NestJS API + BFF-less multi-client SPAs. Binds: all apps. Prevents: duplicate business logic in Next.js route handlers or Zalo client. Rule: clients call REST API only; no server-side domain logic in Next.js beyond auth cookie proxy [ASSUMPTION].
- (decision) Monorepo: pnpm workspaces + Turborepo [ASSUMPTION]. Apps: api, web, admin, zalo-mini-app. Packages: ui, types, config, api-client, utils. Prevents: version drift and duplicated DTOs.
- (decision) Payments: SePay + PayOS (user override). NOT ZaloPay/VNPay/MoMo. Unified PaymentProvider port in API; channel field on Payment (web|zalo). Web + Zalo both use hosted checkout links/redirects from SePay/PayOS. Prevents: PRD provider split and duplicate reconciliation.
- (decision) Payment webhook flow: POST /webhooks/payos and POST /webhooks/sepay; HMAC/signature verify; idempotent PaymentEvent table; activate Subscription on PAID; reconcile job nightly. Prevents: double entitlement and lost webhooks.
- (decision) Auth: NestJS issues JWT access+refresh; AuthIdentity table per PRD; Zalo OAuth server-side token exchange; Google/email via Passport. Zalo Mini App sends zmp getAccessToken to API. Prevents: per-surface session divergence.
- (decision) Zalo Mini App: Vite 5 + zmp-vite-plugin + TanStack Router + zmp-sdk 2.51.x. NOT Next.js. Prevents: Zalo build incompatibilities.
- (decision) Next.js web+admin: App Router (not TanStack Router). TanStack Query + TanStack Form on both. Prevents: routing framework conflict.
- (decision) Excel bulk import: admin uploads → API stores file → BullMQ job validates rows → creates Questions Draft → ImportReport. Max 500/batch per PRD. Prevents: blocking HTTP and partial silent imports.
- (decision) ORM: Prisma + PostgreSQL 16 [ASSUMPTION]. Single migration source in apps/api. Prevents: schema drift.
- (decision) Shared @practice-exam/ui: shadcn/ui + Tailwind tokens from DESIGN.md. Consumed by web, admin, zalo-mini-app where Zalo allows. Prevents: visual drift.
- (decision) Deployment [ASSUMPTION]: api+postgres on VPS/cloud (Railway/Fly/AWS ap-southeast); Next.js on Vercel; Zalo Mini App static bundle via Zalo CDN. ASEAN data residency.
- (decision) User override logged: PRD FR-6/36/42 diverge — payments are SePay/PayOS all channels; ZaloPay config FR-42 replaced by payment provider config screens.
